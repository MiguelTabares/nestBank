## Description

Challenge Bank transfer on Nestjs.

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Environment variables
```bash
DB_CONNECTION= Url MongoDb
DB_HOST= Host
DB_NAME= Name DataBase
DB_USER= User DataBase
DB_PASSWORD= Password DataBase

```
